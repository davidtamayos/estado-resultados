# Dashboard Estado de Resultados

Dashboard ejecutivo de Estado de Resultados para Maquinaria del Humaya.

## Descripción

Este repositorio contiene el dashboard interactivo de Estado de Resultados con análisis mensual de ventas, márgenes y rentabilidad por línea de negocio.

## Acceso

El dashboard está disponible en: https://davidtamayos.github.io/estado-resultados/

## Contenido

- `index.html` - Dashboard interactivo con datos de 6 meses
- - Análisis por línea: Maquinaria, Refacciones, Servicio, Productos Aliados, Ferretería, Sistemas de Riego, AMS, Drones, Seminuevos
 
  - ## Actualización
 
  - El dashboard se actualiza automáticamente cada lunes y viernes a las 8:00 AM.
